# Website Bán Bánh Ngọt

Website bán bánh ngọt được xây dựng bằng PHP thuần với đầy đủ tính năng thương mại điện tử.

## Tính năng chính

### 1. Phân quyền người dùng
- **Admin**: Toàn quyền hệ thống
- **Quản lý**: Quản lý sản phẩm, đơn hàng, người dùng
- **Nhân viên**: Xử lý đơn hàng, cập nhật trạng thái
- **Khách hàng**: Mua sắm, đặt hàng, xem lịch sử

### 2. Quản lý sản phẩm
- Thêm, sửa, xóa sản phẩm
- Quản lý danh mục
- Quản lý hình ảnh sản phẩm
- Quản lý tồn kho
- Giá và khuyến mãi

### 3. Giỏ hàng & Đặt hàng
- Thêm sản phẩm vào giỏ hàng
- Thanh toán đa phương thức
- Theo dõi đơn hàng
- Lịch sử đơn hàng

### 4. Thanh toán
- Tiền mặt khi nhận hàng
- Chuyển khoản ngân hàng
- Ví điện tử (MoMo, VNPay)

### 5. Tìm kiếm & Lọc
- Tìm kiếm sản phẩm
- Lọc theo danh mục, giá
- Sắp xếp sản phẩm

## Cài đặt

### Yêu cầu hệ thống
- PHP 7.4 hoặc cao hơn
- MySQL 5.7 hoặc cao hơn
- Apache/Nginx
- Composer (khuyến nghị)
